### command to run

## Backend

### `npm install`
### `npm run server`

Open [http://localhost:4500](http://localhost:4500) to view it in the browser.
--

## Frontend

### `npm install`
### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
--
